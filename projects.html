Project Portfolio

Here you'll find a deeper look at some of my favourite projects, including technical concepts, design choices, and a gallery of screenshots.

Studio Lens

Concepts Used: HTML5, CSS3, Responsive Design
Summary: Studio Lens is a photography studio website built to practice layout, styling, and basic interactivity. I focused on semantic HTML, modular CSS, and making the site look good on both desktop and mobile. The gallery page uses CSS grid, and the contact form includes basic validation.

Weather App

Concepts Used: JavaScript, REST APIs, Async/Await, Error Handling
Summary: This app fetches live weather data using the OpenWeatherMap API. I learned how to handle asynchronous requests, display loading states, and manage errors gracefully. The UI adapts to different screen sizes and uses custom icons for weather conditions.

Task Tracker

Concepts Used: React, State Management, Local Storage
Summary: Task Tracker is a simple productivity app built with React. It features dynamic state updates, persistent storage using localStorage, and a clean, minimalist design. I practiced component-based architecture and learned how to lift state for shared functionality.
